# 관계 아크: 유웅룡 ↔ 편집장

## 1. 관계 정의

이 관계는 상사와 부하인 동시에,
`접어 둔 기록자`와 `아직 접히기 전 기록자`의 관계다.

## 2. 시작점

- 편집장은 유웅룡의 냉소를 높이 산다
- 유웅룡은 편집장을 현실을 잘 아는 인간으로 본다
- 둘 다 진실보다 `읽히는 방식`을 먼저 안다

## 3. 중간 변화

- 과거 기사 발견 이후 관계가 단순 업무 관계가 아니게 된다
- 유웅룡은 편집장을 보며 자기 미래를 본다
- 편집장은 유웅룡을 보며 접지 않았던 자기 과거를 본다

## 4. 전환점

- 편집장은 끝까지 유웅룡을 완전히 돕지는 않는다
- 하지만 완전히 막지도 못한다
- 이 미묘함이 중요하다

## 5. 마지막 상태

- 적대라기보다 불편한 계승 관계
- 유웅룡이 끝까지 적으려 할 때, 편집장은 자기가 못 한 걸 보는 사람

## 6. 한 줄 요약

유웅룡과 편집장은 `정의로운 대립`이 아니라,
`한 번 접힌 기록이 다음 세대에서 다시 펴질 수 있는가`를 묻는 관계다.
