## 기능 TC

### `F-001` 전체 씬 부트 확인
- 목적: 모든 씬이 `qa_scene` 진입 시 첫 인터랙션까지 정상 도달하는지 확인한다.
- 방법: `qa_fast_batch.js` 또는 `scene_boot_check.js`
- 통과 기준: 첫 상호작용 UI가 `choice` 또는 `evidence`로 도달한다.

### `F-002` 루트 선택지 진행 확인
- 목적: 선택 씬의 루트 선택지가 클릭 후 같은 상태에 갇히지 않는지 확인한다.
- 방법: `qa_fast_batch.js`, 실패 시 `scene_choice_runner.js`
- 통과 기준: 클릭 후 상태 시그니처가 변한다.

### `F-003` 루트 증거 제출 진행 확인
- 목적: 증거 제출 씬의 루트 증거 제출이 제출 후 진행 또는 반응 변화를 만든다.
- 방법: `qa_fast_batch.js`, 실패 시 `evidence_choice_runner.js`
- 통과 기준: 제출 후 상태 시그니처가 변한다.

### `F-004` 선택 씬 심층 choice-only 감사
- 목적: 선택 씬 내부 반복 조사 루프가 숨은 막힘 없이 순환하는지 본다.
- 방법: `scene_choice_audit.js` with `QA_INCLUDE_EVIDENCE=0`
- 통과 기준: 실패 0건 또는 의도된 종료만 존재한다.

### `F-005` 조건부 후반 선택지 노출 확인
- 목적: `ch6_ritual_scene` 등 조건부 선택지가 필요한 상태 주입에서 정상 노출되는지 본다.
- 방법: 상태 주입 + `scene_choice_runner.js`
- 통과 기준: 조건 충족 시 추가 선택지가 보이고 진행된다.

## 데이터/정합성 TC

### `D-001` 구조 검증
- 목적: 장면 참조, 조건, choice_group, emotion 참조가 구조적으로 깨지지 않았는지 확인한다.
- 방법: `validate_game_data.py`
- 통과 기준: 구조상 오류 없음.

### `D-002` 도달 가능성 검증
- 목적: `first_scene` 기준 도달 불가 씬이 없는지 확인한다.
- 방법: `check_story_flow.py`
- 통과 기준: unreachable scene count = 0.

### `D-003` 자산 참조 검증
- 목적: 배경, 초상화, 아이템 이미지, 오디오 경로가 실제 파일과 맞는지 확인한다.
- 방법: 커스텀 경로 검증 스크립트
- 통과 기준: 끊긴 참조가 없다.

### `D-004` 장면 흐름/챕터 점프 검토
- 목적: 씬 전환이 장 단위로 갑자기 비논리적으로 건너뛰지 않는지 확인한다.
- 방법: 그래프 및 메인 경로 검토
- 통과 기준: 의도된 분기 외 비정상 점프 없음.

## 재미/서사 TC

### `N-001` 사망/생존 정합성 검토
- 목적: 이미 죽은 인물이 이후 장면에 재등장할 때 환영, 잔향, 시체, 회상 중 무엇인지 플레이어가 이해 가능한지 본다.
- 대상: `ch2_factory_shock`, `ch2_ipangyu`, `ch5_ritual_room`, `ch5_ipangyu_deal`
- 통과 기준: 생존 오해 없이 연출 의도가 텍스트에서 읽힌다.

### `N-002` 조사 종료 후 다음 동기 검토
- 목적: 조사 씬이 끝났을 때 플레이어가 다음에 왜 이동하는지 납득 가능한지 본다.
- 대상: `ch2_cafe`, `ch3_room4`, `ch2_ipangyu`, `ch6_article`
- 통과 기준: 새 사실 1개와 다음 행동 동기 1개가 남는다.

### `N-003` 후반 결단 차별화 검토
- 목적: 후반 선택지가 서로 다른 리스크와 감정 온도로 읽히는지 본다.
- 대상: `ch5_guarded_door`, `ch5_ipangyu_deal`, `ch6_ritual_scene`
- 통과 기준: 선택 의미가 서로 겹치지 않는다.

### `N-004` 기사 작성 보상 체감 검토
- 목적: 최종 기사 작성이 누적 조사 성과를 집행하는 장면으로 읽히는지 본다.
- 대상: `ch6_article`, `ch6_epilogue`
- 통과 기준: 조사로 얻은 증거와 엔딩 연결 고리가 분명하다.

### `N-005` 감정선 자연스러움 검토
- 목적: 주요 인물의 감정 톤이 씬 사이에서 급변하거나 설명 없이 바뀌지 않는지 확인한다.
- 대상: 유웅룡, 송순, 송금, 이해심, 이판규
- 통과 기준: 감정 변화가 사건 또는 정보 변화에 의해 설명된다.
