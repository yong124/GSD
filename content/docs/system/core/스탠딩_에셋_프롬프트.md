# 스탠딩 에셋 프롬프트

## 목적

현재 경성뎐은 스탠딩 시스템은 들어가 있지만, 실제 이미지는 대부분 기존 portrait를 임시 재사용 중이다.

이 문서는 새 스탠딩 반신/전신 이미지를 제작할 때 바로 참고할 수 있는 작업용 프롬프트 문서다.

---

## 공통 방향

- 시대 배경: 1930년대 경성
- 장르 톤: 미스터리, 조사, 공포, 감응, 기록과 지워짐
- 화면 용도: 비주얼노벨식 스탠딩, 배경 위 좌/중/우 배치
- 기본 구도: 반신 또는 허벅지 위까지 보이는 스탠딩
- 배경: 투명
- 조명: 과한 판타지보다 현실 기반, 단 공명/의식 계열은 미세한 비현실 허용
- 표정: 과장된 만화식보다 절제된 불안, 억눌린 감정, 기이한 분위기 우선
- 용도: 게임 런타임에서 감정별 교체용

---

## 파일명 규칙

기본 규칙:

```text
game/assets/standing/<character>_<emotion>.png
```

예시:

```text
game/assets/standing/yuu_neutral.png
game/assets/standing/yuu_tense.png
game/assets/standing/songsoon_afraid.png
game/assets/standing/ipangyu_crazy.png
```

권장 emotion 이름:

- `neutral`
- `tense`
- `uneasy`
- `afraid`
- `shaken`
- `angry`
- `trance`
- `crazy`

---

## 최우선 제작 대상

### 1. Yuu / 유웅룡

필요 감정:

- `neutral`
- `tense`
- `shaken`
- `trance`

설명:

- 1930년대 경성의 조선인 기자
- 마른 인상, 날카로운 눈매, 쉽게 지치지만 끝까지 물고 늘어지는 사람
- 지나치게 영웅적이지 않고, 늘 피곤과 집요함이 같이 보이면 좋음

기본 프롬프트:

`1930s Gyeongseong Korean male journalist, slim build, sharp tired eyes, vintage suit and coat, half-body standing character, transparent background, realistic painted visual novel style, restrained expression, noir mystery tone`

감정별 포인트:

- `neutral`: 관찰 중, 감정 억제
- `tense`: 의심과 경계
- `shaken`: 충격을 삼키는 상태
- `trance`: 공명에 잠깐 끌린 상태, 시선이 멍하게 풀림

### 2. Songsoon / 송순

필요 감정:

- `neutral`
- `uneasy`
- `afraid`
- `shaken`

설명:

- 낙원 카페의 핵심 인물
- 억눌린 불안, 경계심, 그러나 쉽게 부서지지 않는 단단함이 같이 있어야 함
- 화려한 무희보다는 현실 속 생존자의 분위기

기본 프롬프트:

`1930s Gyeongseong Korean woman, cafe singer or hostess mood, modest period dress, half-body standing character, transparent background, realistic painted visual novel style, anxious but composed, eerie mystery tone`

감정별 포인트:

- `neutral`: 최대한 침착한 얼굴
- `uneasy`: 불안을 숨기는 상태
- `afraid`: 분명한 공포
- `shaken`: 버티지만 흔들린 상태

### 3. Ipangyu / 이판규

필요 감정:

- `neutral`
- `crazy`
- `trance`

설명:

- 병든 광신도이자 사건의 문턱을 열어버린 인물
- 사람처럼 보이되, 이미 절반은 현실에서 이탈한 기색이 중요

기본 프롬프트:

`1930s Korean male prisoner, gaunt face, disturbed eyes, worn institutional clothing, half-body standing character, transparent background, realistic painted horror visual novel style, unsettling religious madness`

감정별 포인트:

- `neutral`: 불안정하지만 아직 말이 통하는 상태
- `crazy`: 광기, 일그러진 웃음, 눈빛 강조
- `trance`: 홀린 듯 멍한 몰입

### 4. Okryeon / 옥련

필요 감정:

- `neutral`
- `afraid`
- `shaken`

설명:

- 낙원의 생존자 느낌
- 겁먹었지만 완전히 무너지지는 않은 상태

기본 프롬프트:

`1930s Korean young woman, modest worker clothing, fearful but grounded, half-body standing character, transparent background, realistic painted visual novel style, dim mystery lighting`

### 5. Haesim / 이해심

필요 감정:

- `neutral`
- `trance`
- `crazy`

설명:

- 의식과 감응의 중심축
- 인간인지 사제인지 경계가 흐려지는 분위기

기본 프롬프트:

`1930s Korean ritual leader, androgynous eerie calm, ceremonial details in period clothing, half-body standing character, transparent background, realistic painted horror visual novel style, occult resonance mood`

### 6. Songgeum / 송금

필요 감정:

- `neutral`
- `trance`

설명:

- 사건의 중심에 놓인 실종 여급
- 현실 인물 같아야 하지만, 이미 어딘가 다른 결로 넘어간 느낌이 필요

기본 프롬프트:

`1930s Korean woman, missing cafe worker, pale and distant expression, half-body standing character, transparent background, realistic painted visual novel style, ghostly but still human`

### 7. Cheonyonghae / 천용해

필요 감정:

- `neutral`

설명:

- 현재는 기본 이미지 자체가 없음
- 최소 1장 필수

기본 프롬프트:

`1930s Korean male, stern presence, period clothing, half-body standing character, transparent background, realistic painted visual novel style, grounded and serious`
