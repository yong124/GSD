# 테이블 완전 새구조

> 범용 HTML 비주얼노벨/ADV 엔진 기준 테이블 설계.
> 경성뎐은 이 구조 위에서 동작하는 첫 번째 타이틀이다.

---

## 설계 원칙

1. **범용성** — 경성뎐 특화 개념(침식, 공명, 기자 등)이 테이블 이름이나 컬럼에 직접 들어가지 않는다.
2. **Flag 없음** — 임의 키-값 플래그를 사용하지 않는다. 모든 상태는 명시적 타입으로 표현한다.
3. **Effect 분리** — 선택지나 대사에 붙는 효과는 독립 테이블로 관리한다.
4. **Choice 확장** — 퍼즐, 증거 제시, 클라이맥스 조합 등 특수 인터랙션은 별도 테이블 없이 Choice 구조를 확장해서 처리한다.
5. **Gauge 통합** — 수치 기반 상태(침식, 조사도 등)는 GaugeTable로 일원화한다. 단 캐릭터별 신뢰도(Trust)는 특수 케이스로 별도 처리한다.

## 테이블 목록

| 테이블 | 분류 | 역할 |
|---|---|---|
| `GaugeTable` | 신규 | 게임 내 수치 정의 |
| `GaugeStateTable` | 신규 | 수치 단계 및 진입 효과 정의 |
| `EffectTable` | 신규 | 효과 묶음 정의 |
| `ConditionTable` | 수정 | 공통 조건 판정 |
| `SceneTable` | 수정 | 씬 메타 |
| `DialogTable` | 수정 | 씬 내부 대사 노드 |
| `ChoiceGroupTable` | 수정 | 선택지 묶음 |
| `ChoiceTable` | 수정 | 개별 선택지 |
| `BranchTable` | 유지 | 씬 종료 후 자동 분기 |
| `EvidenceTable` | 유지 | 정적 증거 정의 |
| `EvidenceCategoryTable` | 유지 | 단서 UI 카테고리 |
| `CharacterTable` | 수정 | 캐릭터 기본/수첩 정보 |
| `CharacterEmotionTable` | 유지 | 감정별 이미지 |
| `InvestigationTable` | **제거** | SceneTable + ChoiceGroupTable로 흡수 |

## 핵심 구조 변경 요약

### Flag 제거

기존의 임의 플래그(`CalledEditor: true`, `SolvedQuestionCount: 3` 등)를 모두 제거한다.
대체 타입은 아래와 같다.

| 기존 Flag 용도 | 대체 |
|---|---|
| 행동 여부 (`CalledEditor`) | `ChoiceSelected` |
| 증거 보유 (`HasEvidence_xxx`) | `EvidenceOwned` |
| 누적 수치 (`SolvedQuestionCount`) | `GaugeValue` |
| 씬 방문 여부 | `SceneVisited` |

### Choice 확장

퍼즐, 증거 제시, 클라이맥스 조합 등을 별도 테이블 없이 Choice로 처리한다.

| 인터랙션 | 처리 방식 |
|---|---|
| 일반 선택지 | `ChoiceGroup.AnswerType: Text` |
| 증거 제시 (논리형 퍼즐) | `ChoiceGroup.AnswerType: Evidence` |
| 탐색 선택지 | `ChoiceGroup.AnswerType: Text` + 씬 설계 |
| 클라이맥스 조합 | `ChoiceGroup.AnswerType: Evidence` + 씬 설계 |

`AnswerType: Evidence`일 때 UI는 텍스트 버튼 대신 현재 보유한 증거 인벤토리 전체를 표시한다.

### Effect 분리

기존에 ChoiceTable에 직접 붙어있던 효과 컬럼들을 EffectTable로 분리한다.
ChoiceTable과 DialogTable 모두 `EffectGroupID`로 참조한다.

### InvestigationTable 제거

| 기존 컬럼 | 이동 위치 |
|---|---|
| `Title`, `Hint` | `SceneTable.InvestigationTitle`, `SceneTable.InvestigationHint` |
| `Budget` | `ChoiceGroupTable.MaxSelectable` |
| `ChoiceGroupID` | `DialogTable.ChoiceGroupID`로 직접 참조 |
