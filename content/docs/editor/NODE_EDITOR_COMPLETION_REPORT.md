# 경성뎐 Node Editor 완료 보고

## 1. 작업 목적

본 작업의 목표는 `EditorNode`를 `경성뎐` 프로젝트의 메인 기획 작업 툴로 사용할 수 있는 수준까지 끌어올리는 것이었다.

기준은 다음과 같았다.

- 기본 편집 안정화
- 대사 프리뷰 지원
- 구조 검수 기능 확보
- 데이터 구조 일치성 확보
- 검색 / 필터 / 기획 검수 기능 추가
- 준대량 편집 기능 추가
- generated xlsx 개선
- 시각화 고도화
- 운영 / 포트폴리오 문서 정리

---

## 2. 완료 범위

### 2-1. 에디터 기능

- 씬 생성 / 삭제 / 복제 / 이름 변경
- Undo / Redo
- 대사 / 선택지 / 분기 / 단서 편집
- 대사 프리뷰
- 연결 드래그 편집
- 선택지 `next_dialogue` / 대사 `condition` / 분기 `order` 반영

### 2-2. 검수 기능

- 끊긴 씬 연결 검출
- 선택지 / 분기 연결 누락 검출
- 미참조 씬 검출
- 중복 `EvidenceID` 검출
- 중복 `Label` 검출
- 현재 씬 기준 읽는 플래그 / 쓰는 플래그 표시
- 이전 연결 / 다음 연결 표시

### 2-3. 탐색 / 운영 기능

- 씬 / 라벨 / 플래그 / 단서 검색
- 챕터 필터
- 배경 일괄 변경
- 스타일 일괄 변경
- 플래그 키 전역 치환
- 문자열 일괄 치환
- 챕터 기준 정렬 보조

### 2-4. 시각화

- 챕터별 노드 색상
- 엔딩 노드 강조
- 미참조 씬 시각 경고
- 줌 UI
- 미니맵
- 자동 정렬

### 2-5. 데이터 / 문서

- `EditorNode`와 `export_to_json.py` 구조 정렬
- generated xlsx 메타 / 요약 / 시트 선택 / csv-tsv 출력 지원
- 시스템 / 에디터 / 포트폴리오 문서 세트 정리
- `content` 폴더 구조 재정리

## 5. 한 줄 정리

`EditorNode`는 이제 `경성뎐`의 메인 기획 작업 툴로 사용할 수 있는 수준까지 정리되었고, xlsx는 대량 편집과 검수 시에만 보조적으로 사용하는 구조가 갖춰졌다.
