# 경성뎐 Node Editor 사용 가이드

## 1. 문서 목적

본 문서는 `EditorNode`를 메인 작업 툴로 사용할 때 필요한 기본 사용법을 정리한 가이드다.

---

## 2. 기본 역할

`EditorNode`는 다음 작업을 담당한다.

- 씬 생성 / 삭제 / 복제
- 씬 제목, ID, 챕터, 배경, 음악, 이펙트 수정
- 대사 / 선택지 / 분기 / 단서 편집
- 씬 간 연결 편집
- 구조 검수
- 대사 프리뷰 확인

---

## 3. 기본 작업 순서

1. 씬을 선택한다.
2. 패널에서 씬 메타 정보를 수정한다.
3. 대사 / 선택지 / 분기 / 단서를 추가하거나 수정한다.
4. 노드 연결과 검수 패널을 확인한다.
5. `내보내기`로 `game_data.js`를 갱신한다.

---

## 4. 주요 기능

### 4-1. 대사 프리뷰

- 현재 편집 중인 대사 행을 바로 미리 볼 수 있다.
- 스타일과 배경이 함께 반영된다.
- `이전 / 다음` 버튼으로 같은 씬 안의 대사를 넘겨볼 수 있다.

### 4-2. 구조 검수

- 끊긴 씬 연결
- 끊긴 선택지 연결
- 끊긴 분기 연결
- 중복 `EvidenceID`
- 중복 `Label`
- 미참조 씬

위 항목을 패널에서 바로 확인할 수 있다.

### 4-3. 탐색 / 검색

- 씬 ID 검색
- 씬 제목 검색
- 플래그 키 검색
- 라벨 검색
- 단서 ID 검색
- 챕터 필터

### 4-4. 준대량 편집

- 필터된 씬 배경 일괄 변경
- 필터된 씬 대사 스타일 일괄 변경
- 플래그 키 전역 치환
- 문자열 일괄 치환
- 챕터 기준 정렬 보조

### 4-5. 시각화

- 챕터별 노드 색상
- 엔딩 노드 강조
- 미참조 씬 표시
- 미니맵
- 줌 UI
- 자동 정렬 버튼

---

## 5. 연결 편집 방식

- 노드 우측 핀에서 드래그해 다른 노드의 입력 핀으로 연결한다.
- 기본 연결, 선택지 연결, 분기 연결을 각각 별도로 다룬다.
- 우클릭하면 현재 연결을 제거할 수 있다.

---

## 6. 단축키

- `Ctrl+S`: 내보내기
- `Ctrl+Z`: 실행 취소
- `Ctrl+Y` 또는 `Ctrl+Shift+Z`: 다시 실행
- `Delete`: 선택 씬 삭제
- `Esc`: 선택 해제

---

## 7. 주의 사항

- `game_data.js`는 직접 수정하지 않는다.
- `script.xlsx`는 자동 덮어쓰기하지 않는다.
- 구조 수정은 가능한 한 Node Editor에서 처리한다.
- xlsx는 대량 편집이나 검수 시에만 보조적으로 사용한다.

---

## 8. 한 줄 정리

Node Editor는 `경성뎐`의 메인 기획 작업 툴이며, 구조 수정과 기본 검수는 가능한 한 이 안에서 끝내는 것을 원칙으로 한다.
