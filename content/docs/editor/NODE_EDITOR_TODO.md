# 경성뎐 Node Editor TODO

본 문서는 `EditorNode`를 메인 작업 툴로 운영하기 위해 필요한 개선 항목을 우선순위 기준으로 정리한 TODO 문서다.

운영 전제는 다음과 같다.

- 메인 편집 툴은 `EditorNode`
- `xlsx`는 대량 편집 / 검수 / 공유가 필요할 때만 사용
- 구조 수정은 가능한 한 에디터에서 끝내는 것을 목표로 함
