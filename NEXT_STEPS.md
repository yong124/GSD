## 1️⃣ 긴급 (24시간 내)

### A. AI 이미지 생성 & 교체
**작업**: `game/data/prompts.md`의 15개 프롬프트를 이미지 생성 AI에 입력
- **배경** (8개): court, newsroom, hospital, factory, cafe, storage, slum, underground
- **초상화** (5개): 유웅룡, 송순, 편집장, 이판규, 이해심
- **단서** (2개): 쪽지, 시신

**생성 후 저장**:
```
game/assets/bg/*.jpg        (권장 1280×720)
game/assets/portraits/*.png (권장 220×280)
game/assets/ev/*.jpg        (권장 300×200)
```

**코드 수정 없음** — 자동으로 적용됨

---

### B. 브라우저 테스트
**도구**: Chrome 또는 Firefox (최신)
**경로**: `http://localhost:3900` 또는 파일로 직접 열기
**체크리스트**:
- [ ] 타이틀 스크린 (New Game / Continue 버튼)
- [ ] Chapter 1 시작 (법정 씬, 대화 진행)
- [ ] 선택지 클릭 (플래그 기록 확인)
- [ ] 단서 메모장 (📋 버튼, 자동 수집)
- [ ] 저장/불러오기 (상단 버튼)
- [ ] 모든 6개 챕터 진행 가능
- [ ] 3개 엔딩 분기 (A/B/C) 모두 도달 가능

## 2️⃣ 중요 (1주일 내)

### C. 최종 시나리오 검증
**확인 사항**:
