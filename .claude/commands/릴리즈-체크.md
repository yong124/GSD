# 릴리즈 체크

이 명령은 작업 마감 전 검증, 임시 산출물 정리, handoff 정리, 커밋 범위 점검에 쓴다.

## 기본 순서

1. `git status --short`로 워크트리를 본다.
2. 변경 축에 맞는 검증만 정확히 돌린다.
3. 브라우저 자산을 건드렸다면 `?v=` 누락 여부를 확인한다.
4. generated 또는 handoff 갱신 필요 여부를 판단한다.
5. QA 중간 산출물이나 임시 폴더를 정리한다.
6. 커밋 범위를 한 줄로 설명 가능한지 점검한다.
