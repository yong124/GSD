# 데이터 폴리시

이 명령은 서사 문장, 분기 감정선, 선택 후 반응 대사처럼 `game_data.js` 중심의 데이터 다듬기 작업에 쓴다.

## 기본 순서

1. `git status --short`로 현재 워크트리를 확인한다.
2. 한글이 깨져 보이면 UTF-8부터 확인한다.
3. 필요한 장면의 대사, 분기, 반응 대사를 정리한다.
4. 구조 필드는 유지한 채 텍스트와 흐름만 다듬는다.
5. `validate_game_data.py`를 돌린다.
6. 필요하면 generated xlsx를 갱신한다.
