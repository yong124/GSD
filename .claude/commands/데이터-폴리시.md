# 데이터 폴리시

이 명령은 아래 순서로 움직인다.

1. `game_data.js`를 UTF-8로 확인
2. 관련 장면 대사/분기/반응 대사 수정
3. 필요 시 `next_dialogue`, `priority_*`, 단서 반응도 함께 조정
4. `validate_game_data.py` 실행
5. 필요하면 generated xlsx 갱신
6. 바뀐 장면 / 검증 / 남은 후속 작업을 짧게 요약

우선 사용할 스킬:

- `gsd-data-polish`

필요 시 같이 사용할 스킬:

- `gsd-priority-investigation`
- `gsd-release-checks`
