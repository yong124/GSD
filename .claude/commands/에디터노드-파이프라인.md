# 에디터노드 파이프라인

이 명령은 EditorNode, export 스크립트, validate, generated xlsx, 테이블 스키마처럼 저작 도구와 데이터 파이프라인이 함께 얽힌 작업에 쓴다.

## 기본 순서

1. 변경이 에디터 전용인지, 파이프라인 전용인지, 스키마 전체인지 먼저 정한다.
2. 스키마 전체면 런타임 기대값부터 확인한다.
3. EditorNode 직렬화와 정규화를 맞춘다.
4. export 또는 generated 흐름을 맞춘다.
5. validate와 문서를 함께 갱신한다.
6. Python과 EditorNode JS 검증을 돌린다.
